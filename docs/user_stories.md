## Phase 2 — 已完成

| ID | 模块 | 简介 | User Story | 验收标准 (AC) | 优先级 | 状态 |
|---|---|---|---|---|---|---|
| DT-C8 | 组件 | 行增删 | 作为用户，我需要在表格中增加新行和删除已有行，以便管理数据记录。 | 1. 点击"添加行"按钮，在表格底部插入一条空行并自动进入编辑态。<br>2. 选中行后点击"删除"按钮或右键菜单可删除行，需二次确认。<br>3. 增删操作通过 Yjs 实时同步至协作者。<br>4. 支持批量选中多行后一次性删除。 | P0 | ✅ Done |
| DT-C9 | 组件 | 列交互管理（列宽/顺序/隐藏/冻结/自动列宽） | 作为用户，我需要灵活调整列的宽度、顺序、可见性和冻结状态，以打造适合自己工作流的数据视图。 | 1. 拖拽列头右边缘可调整列宽，释放后生效，最小宽度 ≥ 60px。<br>2. 拖拽列头可调整列顺序，拖拽时显示占位指示器。<br>3. 列头右键菜单提供"隐藏列"选项；工具栏提供列可见性管理面板（Checkbox 列表）。<br>4. 列头右键菜单提供"冻结至此列"选项，冻结列固定在左侧不随水平滚动。<br>5. 工具栏提供"自动列宽"按钮，点击后根据当前可见行内容计算最佳列宽。<br>6. 列宽/顺序为本地视图状态，不触发协同同步。 | P0 | ✅ Done |
| DT-C10 | 组件 | 条件高亮（Conditional Formatting） | 作为用户，我需要根据条件自动高亮单元格，以便快速识别关键数据。 | 1. 工具栏提供"条件格式"入口，打开配置面板。<br>2. 支持配置规则：选择目标列 → 选择条件（大于/小于/等于/包含/为空等） → 设置高亮颜色。<br>3. 支持添加多条规则，规则按优先级从上到下生效；冲突时取最高优先级。<br>4. 已启用的条件格式规则在对应列 Header 区域显示指示器（如彩色圆点）。<br>5. 条件格式为本地视图状态。 | P1 | ✅ Done |
| DT-C11 | 组件 | 视图配置持久化 | 作为用户，我需要在刷新页面后恢复上次的筛选、分组、列宽等视图配置，避免每次重新设置。 | 1. 持久化范围：filter / group / 列宽 / 列顺序 / 列隐藏 / col freeze / conditional highlight 配置。<br>2. 配置以 `roomId + userId` 为 key 存储到 `localStorage`。<br>3. 页面加载时自动读取并应用已保存的视图配置。<br>4. 提供"重置视图"按钮，清除所有已存储配置并恢复默认。 | P1 | ✅ Done |
| DT-C12 | 组件 | 键盘导航 | 作为用户，我需要使用方向键在单元格之间移动选中焦点，以提高键盘操作效率。 | 1. 选中 cell 后，↑↓←→ 方向键移动选中焦点到相邻 cell。<br>2. Tab 键向右移动到下一个可编辑 cell，Shift+Tab 反向移动。<br>3. Enter 键进入当前 cell 编辑态；编辑态下 Enter 提交并移动到下一行同列，Escape 取消编辑。<br>4. 移动到可视区域外时自动触发滚动以确保目标 cell 可见。 | P1 | ✅ Done |
| DT-C13 | 组件 | 数值格式化显示 | 作为用户，我需要对 number 列设置数字显示格式（小数位数、千分位、货币符号），以便数据展示更加清晰。 | 1. 列定义支持 `numberFormat` 配置项：`{ decimals?: number; thousandSeparator?: boolean; currency?: string }`。<br>2. 查看态按格式渲染（如 `1234.5` → `¥1,234.50`）。<br>3. 编辑态显示原始数值，退出编辑后恢复格式化展示。<br>4. 排序和筛选基于原始数值，不受格式化影响。 | P1 | ✅ Done |
| DT-C14 | 组件 | 公式计算列 | 作为用户，我需要创建基于其他列的公式计算列（四则运算），以便自动汇总和派生数据。 | 1. 列定义支持 `formula` 属性，值为字符串表达式（如 `"col_a + col_b * 0.1"`），引用其他列 ID。<br>2. 公式列为只读，值在渲染时实时计算。<br>3. 被引用列数据变更时，公式列自动重新计算。<br>4. 鼠标 Hover 到公式列 Header 时，Tooltip 显示具体计算公式。<br>5. 支持 `+` `-` `*` `/` 四则运算和括号，不支持聚合函数。<br>6. 公式引用的列被删除或隐藏时，公式列显示 `#REF!` 错误。 | P2 | ✅ Done |
